Installation

First download and install Python 3 from the official website (on Linux any modern Python 3 will do).

Then download the latest Simpyt.pyz release.

And finally run (double click) the simpyt.pyz file, and go to http://127.0.0.1:9999 in your PC web browser, or to http://YOUR_PC_IP_ADDRESS:9999/ from a cellphone or tablet.

That's it! You don't even need to install anything in your phone or tablet, just the web browser.
